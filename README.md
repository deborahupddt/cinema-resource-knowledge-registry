# cinema-resource-knowledge-registry
本仓库用于系统整理影视类网站在公开网络环境中所呈现的基础资料与结构信息，重点记录首页栏目分布方式、分类层级逻辑、标签体系构建形式以及页面标题组织结构。通过持续性的文字归档方式建立长期可更新的资料档案，用于观察不同影视平台在页面设计与内容组织方面的变化趋势。所有资料均来源于公开访问页面，仅作为信息整理与研究参考使用。本项目不包含任何影视资源文件，不涉及播放、下载或分发服务，仅用于公开信息结构记录与对比分析用途存在。
